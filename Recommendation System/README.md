# Recommendation System

This project is a simple recommendation system that suggests movies to users based on their preferences.

## Technique Used
This system uses **content-based filtering**, where recommendations are generated based on user-selected genres.

## Technologies Used
- Python

## Features
- Takes user preferences as input (genres)
- Filters movies based on preferences
- Sorts recommendations using ratings
- Displays top recommended movies

## How to Run
```bash
python recommendation.py
